# Initia Expense Tracker 📊

A sophisticated financial management and predictive analytics tool designed specifically for students and users within the **Initia ecosystem**. This project bridges the gap between daily expense tracking and advanced statistical forecasting.

## 🚀 Key Features
* **Predictive Spending:** Uses time-series modeling to forecast future expenses based on historical data.
* **Financial Runway:** Calculates how long your current balance will last using predicted volatility.
* **Initia Integration:** Built to handle ecosystem-specific transactions and student budgeting needs.
* **Mobile-Ready:** Developed and tested in a mobile-first environment (Termux/Acode).

## 📈 Statistical Methodology
Unlike standard trackers that use simple averages, this tool employs a dual-model approach to provide professional-grade insights:

1.  **ARIMA (AutoRegressive Integrated Moving Average):** Used to capture the linear trends and seasonality in daily spending habits.
2.  **GARCH (Generalized Autoregressive Conditional Heteroskedasticity):** Implemented to model "spending shocks" and volatility, accounting for irregular or large one-time purchases.

## 🛠️ Tech Stack
* **Language:** JavaScript / Node.js
* **Backend Framework:** Express.js
* **Statistical Logic:** Custom implementations of ARIMA/GARCH frameworks.
* **Development Environment:** Termux & Acode (Mobile Development Workflow)
